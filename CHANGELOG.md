# Changelog

All notable Jetstream89 changes are recorded here. The project follows a simple semantic-versioning direction while it remains a prototype.

## [2.0.0] - 2026-07-21

### Added

- EN/ES browser-local interface layer
- Flight Crew profiles for Jack Rodríguez and Kevin Cusnir
- original Web Audio station identifiers with explicit user activation
- original neon flightdeck SVG artwork
- radar-style winged-microphone logo with integrated `89`
- Merch Lab concept page with local interest saving and no fake checkout
- Station Studio with contrast, motion, density and privacy controls
- integration-readiness dashboard and public version history
- PWA shortcuts for Schedule, Flight Crew and Station Studio
- richer host, merch and studio sections on the home page

### Changed

- redesigned home page with a premium music-platform-inspired cockpit
- upgraded Schedule, Rock Stories and Rock Atlas navigation and presentation
- expanded README with v2 architecture, rights, privacy and deployment notes
- strengthened GitHub Actions checks for both JavaScript files, required v2 assets, HTML, local links and brand consistency

### Honest limitations

- no synchronized Jetstream89 live stream is connected
- Spotify playback remains inside official embeds
- schedule, votes, requests and merchandise are still prototype data or local interactions
- no backend, accounts, checkout, inventory or payment collection exists
- most editorial paragraphs remain English-first while the shared interface layer supports EN/ES

## [1.1.0] - 2026-07-20

### Added

- Spotify/TIDAL-inspired static cockpit
